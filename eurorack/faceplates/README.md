# Hog Moduleur / Eurorack Faceplates

3D printable (or lasercut) faceplates designed for taking the Moduleur apart and use in Eurorack format. 

## Components

1. [VCO](https://github.com/shmoergh/moduleur/tree/main/eurorack/faceplates/eurorack-panel-shmoergh-moduleur-vco.step)
2. [VCF](https://github.com/shmoergh/moduleur/tree/main/eurorack/faceplates/eurorack-panel-shmoergh-moduleur-vcf.step)
3. [SIDECHAIN + MIXER](https://github.com/shmoergh/moduleur/tree/main/eurorack/faceplates/eurorack-panel-shmoergh-moduleur-sidechain-mixer.step)
4. [ENVELOPE + VCA](https://github.com/shmoergh/moduleur/tree/main/eurorack/faceplates/eurorack-panel-shmoergh-moduleur-envelope-vca.step)
5. [CRUSH + LFO + OUTPUT](https://github.com/shmoergh/moduleur/tree/main/eurorack/faceplates/eurorack-panel-shmoergh-moduleur-crush-lfo-output.step)
6. [BRAIN](https://github.com/shmoergh/moduleur/tree/main/eurorack/faceplates/eurorack-panel-shmoergh-moduleur-brain.step)
